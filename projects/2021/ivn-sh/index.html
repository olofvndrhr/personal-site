ivn.sh

link to repo: https://github.com/olofvndrhr/ivn.sh-conf

I was unsure on how to create and deploy my personal site, without too much hassle. So I landed on the pretty popular static site framework hugo. It was relatively simple to set up and get working.

For the deployment, I went with cloudflare pages for “hosting” and web analytics for the analytics part.

I choose this option because I already manage all my domains via cloudflare, and so everything is on one platform for simpler management.

Made using anemone Zola theme
